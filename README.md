# Python data visualization
This repository contains the following Python files:
* Code to plot yearly GDP data on a world map by country code.
* Code to plot cancer risk data on a USA map.
* Code to plot yearly GDP data on a world map by country name.
* Code to merge CSV files that contain cancer risk data from the United States.
* Code to plot yearly GDP data by countries using paygal.
* Code that reads an SVG map and writes a CSV file with coordinates calculated using the path data.
* Code to read a map using matplotlib and add scatter points to it.
